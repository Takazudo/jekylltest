#jekylltest

a very simple example of [jekyll](https://github.com/mojombo/jekyll)

* [demo](http://takazudo.github.com/jekylltest/)


##how to use this

```bash
$ jekyll
```

publishes everything from source

```bash
$ jekyll --server
```

then open http://localhost:4000 for debug
